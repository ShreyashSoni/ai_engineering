# AI & LLM Engineering
A repo to store the code for my AI experiments and projects.

## Setup
1. run `uv sync` to create the virtual environment for the projects
2. create a `.env` file and add your OpenAI and Google Gemini API keys. (`OPENAI_API_KEY` and `GOOGLE_API_KEY`)
